Tambahkan daftar 14 syuhada (sahabat yang gugur) Perang Badr ke lokasi **`makam-syuhada-badr`**, dengan konteks singkat tiap nama. Sumber: Sirah Ibnu Hisyam, Ibnu Ishaq, Ibnu Katsir.

**PENTING karakter:** tulis ﷺ, ’, — sebagai karakter LANGSUNG. JANGAN pakai `\uXXXX` atau `\\uXXXX` (bikin teks ngawur). File `syuhada-badr.ts` sudah memakai karakter benar.

```ts

  nama: string;
  golongan: string;     // 'Muhajirin' | 'Anshar (Aus)' | 'Anshar (Khazraj)'
  konteks: string;
}

export interface Lokasi {
  // ... field yang sudah ada ...
  syuhadaPengantar?: string;
  syuhada?: SyuhadaItem[];
}
```

## LANGKAH 2 — Isi data di objek `id: 'makam-syuhada-badr'` (data/lokasi.ts)

Tambahkan field `syuhadaPengantar` dan `syuhada` (14 item) dari file `syuhada-badr.ts`. Ringkasan:

**Pengantar:** "Dalam Perang Badr (17 Ramadhan 2 H / 624 M), 14 sahabat gugur sebagai syuhada — 6 dari Muhajirin dan 8 dari Anshar (2 Aus, 6 Khazraj). Berikut nama-nama mereka menurut riwayat ulama sirah seperti Ibnu Hisyam, Ibnu Ishaq, dan Ibnu Katsir."

**14 Syuhada:**
- **Muhajirin (6):** Ubaidah bin Al-Harits, Umair bin Abi Waqqash, Dzusy Syimalain (Umair bin Abdu Amr), Aqil bin Al-Bukair, Mihja’ bin Shalih, Shafwan bin Baidha’.
- **Anshar–Aus (2):** Sa’ad bin Khaitsamah, Mubasysyir bin Abdul Mundzir.
- **Anshar–Khazraj (6):** Yazid bin Al-Harits (Ibnu Fushum), Umair bin Al-Humam, Rafi’ bin Al-Mu’alla, Haritsah bin Suraqah, Auf bin Al-Harits, Muawwidz bin Al-Harits.

> Teks konteks lengkap tiap nama ada di file `syuhada-badr.ts` — salin apa adanya (sudah pakai karakter Arab/tanda baca yang benar).

## CATATAN AKURASI (review sebagai mahasiswa Ushuluddin)
- Jumlah 14 syuhada (6 Muhajirin + 8 Anshar) adalah pendapat masyhur dari Ibnu Hisyam & Ibnu Katsir. Ada sedikit perbedaan ejaan/nisbah nama antar kitab — redaksi sudah dibuat sesuai yang umum.
- Beberapa nama punya kisah masyhur (Umair bin Al-Humam dgn kurma, Haritsah bin Suraqah dgn Firdaus tertinggi, Mihja’ sebagai syahid pertama) — sudah disertakan ringkas. Verifikasi bila ingin menambah detail.
- Tambahkan ﷺ hanya untuk Nabi Muhammad; sahabat pakai "radhiyallahu 'anhu" bila ingin (opsional, agar tidak terlalu panjang di UI bisa dihilangkan).
