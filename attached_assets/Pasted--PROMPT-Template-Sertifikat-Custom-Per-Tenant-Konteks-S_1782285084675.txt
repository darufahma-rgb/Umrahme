# PROMPT — Template Sertifikat Custom Per Tenant

## Konteks
Sistem sertifikat sekarang pakai desain hardcoded (gold/dark).
Kita mau admin bisa upload **gambar template sertifikat** (JPG/PNG landscape) per tenant.
Gambar itu jadi background, lalu data jamaah di-overlay di atasnya pakai CSS positioning.

---

## LANGKAH 1 — SQL (jalankan di Supabase SQL Editor dulu)

```sql
ALTER TABLE tenants ADD COLUMN IF NOT EXISTS sertifikat_template_url TEXT;
```

### 2b. Tambah function upload (di bawah `uploadHeroImage`)
```ts
export async function uploadSertifikatTemplate(file: File): Promise<string> {
  const ext = file.type === 'image/png' ? 'png' : file.type === 'image/webp' ? 'webp' : 'jpg';
  const filename = `sertifikat/${crypto.randomUUID()}.${ext}`;
  const { data, error } = await supabase.storage
    .from('logos')
    .upload(filename, file, { upsert: false, contentType: file.type });
  if (error) throw new Error(error.message);
  const { data: { publicUrl } } = supabase.storage.from('logos').getPublicUrl(data.path);
  return publicUrl;
}
```

### 3d. Handler upload
```ts
function handleSertifikatChange(e: ChangeEvent<HTMLInputElement>) {
  const file = e.target.files?.[0];
  if (!file) return;
  if (file.size > MAX_SERTIFIKAT_SIZE) { setError('Ukuran template sertifikat maksimal 5MB.'); return; }
  const allowedTypes = ['image/png', 'image/jpeg', 'image/webp'];
  if (!allowedTypes.includes(file.type)) {
    setError('Template sertifikat harus PNG, JPG, atau WEBP.');
    return;
  }
  setSertifikatFile(file);
  const reader = new FileReader();
  reader.onload = ev => setSertifikatPreview(ev.target?.result as string);
  reader.readAsDataURL(file);
}
```

### Ganti `backgroundColor` di fungsi `unduh` dan `bagikan`

Cari:
```ts
backgroundColor: '#130a04',
```

Ganti jadi:
```ts
backgroundColor: hasTemplate ? '#ffffff' : '#130a04',
```

(Ada 2 tempat — di fungsi `unduh` dan `bagikan`)

---

## Catatan
- Kalau `sertifikat_template_url` null → fallback ke desain gold/dark lama (tidak ada breaking change)
- `crossOrigin="anonymous"` di `<img>` penting supaya `toPng` (html-to-image) bisa capture gambar dari Supabase Storage
- Pastikan Supabase Storage bucket `logos` sudah public (sudah ada dari sebelumnya)
- Nama jamaah di-render di tengah canvas dengan `text-shadow` supaya terbaca di atas template apapun
- Tidak perlu ubah fungsi `unduh` / `bagikan` selain `backgroundColor`
