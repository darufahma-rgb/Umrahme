## LANGKAH 1 — Update Pengelompokan Kota

File: `artifacts/umrahme/src/data/lokasi.ts`

Thaif satu provinsi dengan Makkah (~90km), Badr dekat Madinah (~130km).
Kelompokkan Thaif ke tab Makkah, Badr ke tab Madinah — sama seperti Arafah yang masuk grup Makkah.

Cari fungsi `lokasiByKota`:
```ts
export function lokasiByKota(kota: KotaFilter): Lokasi[] {
  return daftarLokasi.filter((l) => {
    if (kota === 'Makkah') return l.kota === 'Makkah' || l.kota === 'Arafah';
    return l.kota === 'Madinah';
  });
}
```

Ganti jadi:
```ts
export function lokasiByKota(kota: KotaFilter): Lokasi[] {
  return daftarLokasi.filter((l) => {
    if (kota === 'Makkah') return l.kota === 'Makkah' || l.kota === 'Arafah' || l.kota === 'Thaif';
    return l.kota === 'Madinah' || l.kota === 'Badr';
  });
}
```

> Badge kota tetap menampilkan "Thaif"/"Badr" aslinya (karena `l.kota` dipakai untuk badge),
> tapi filter tab mengelompokkannya ke Makkah/Madinah. Tidak perlu ubah type `KotaFilter`.

## LANGKAH 3 — Pastikan Badge Kota Mendukung Warna Baru (opsional)

File: `artifacts/umrahme/src/pages/Peta.tsx`

Cek komponen `CityBadge` di `LokasiDetail.tsx` dan badge kota di `Peta.tsx`.
Saat ini hanya handle 'Makkah' & 'Arafah' khusus, sisanya default. Thaif & Badr
akan otomatis pakai style default — itu sudah cukup. **Tidak wajib diubah.**

Kalau mau kasih warna khusus (opsional), di `CityBadge` (LokasiDetail.tsx) bisa tambah:
```ts
const isThaif = kota === 'Thaif';
const isBadr = kota === 'Badr';
```
lalu sesuaikan style. Tapi ini opsional — default sudah berfungsi.

## Catatan Penting
- Koordinat sudah diverifikasi:
  - Masjid Abdullah bin Abbas Thaif: 21.2701, 40.4082 (Evendo, alamat Wadi Waj Road)
  - Masjid Addas: ~2.5km SW (perkiraan, sumber sirah)
  - Situs Perang Badr: 23.7333, 38.7667 (latitude.to)
  - Makam Syuhada Badr: ~550m barat situs (Madain Project)
- Pastikan karakter Arab & emoji ﷺ/رضي الله عنه ter-encode UTF-8 dengan benar
- Field `gambar: ''` dikosongkan — bisa diisi nanti kalau ada foto
- Jangan ubah lokasi lain yang sudah ada
- Field `kota` di type `Lokasi` (`types/index.ts`) sudah bertipe `string` — TIDAK perlu ubah type, "Thaif" & "Badr" otomatis valid
