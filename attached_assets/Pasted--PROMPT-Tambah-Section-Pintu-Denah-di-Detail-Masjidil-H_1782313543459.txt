# PROMPT: Tambah Section "Pintu & Denah" di Detail Masjidil Haram & Masjid Nabawi

**File terkait:**
- `artifacts/umrahme/src/types/index.ts` (tambah field di interface Lokasi)
- `artifacts/umrahme/src/data/lokasi.ts` (isi data pintu untuk 2 masjid)
- `artifacts/umrahme/src/pages/LokasiDetail.tsx` (render section baru)

Tambahkan informasi pintu-pintu penting + link ke peta/denah resmi untuk Masjidil Haram dan Masjid Nabawi. Sumber: General Authority for the Care of the Two Holy Mosques (alharamain.gov.sa).

---

## LANGKAH 1 — Tambah field di interface `Lokasi` (types/index.ts)

Tambahkan dua field opsional pada interface `Lokasi`:
```ts
export interface PintuMasjid {
  nama: string;        // mis. "Bab as-Salam (Gate 1)"
  fungsi: string;      // deskripsi singkat fungsi/keutamaan pintu
}

export interface Lokasi {
  // ... field yang sudah ada ...
  pintu?: PintuMasjid[];      // daftar pintu penting (untuk masjid besar)
  petaResmiUrl?: string;      // link ke peta/denah resmi
}
```

## LANGKAH 3 — Render section "Pintu & Denah" di LokasiDetail.tsx

Tambahkan section ini di halaman detail (hanya tampil jika `lokasi.pintu` ada), letakkan setelah section Adab atau sebelum Catatan:

```tsx
{lokasi.pintu && lokasi.pintu.length > 0 && (
  <section className="mt-6">
    <div className="mb-3 flex items-center justify-between">
      <div>
        <p className="font-mono text-[8.5px] uppercase tracking-[0.18em] text-mute">Navigasi</p>
        <h3 className="mt-0.5 text-[15px] font-bold text-ink">Pintu Penting & Denah</h3>
      </div>
    </div>

    {/* Daftar pintu */}
    <div className="space-y-2">
      {lokasi.pintu.map((p, i) => (
        <div key={i} className="rounded-xl border border-hairline bg-surface-card p-3">
          <div className="flex items-start gap-2.5">
            <div className="mt-0.5 flex h-6 w-6 flex-none items-center justify-center rounded-md bg-gold/15">
              <IconPeta className="h-3.5 w-3.5 text-gold" />
            </div>
            <div className="min-w-0">
              <h4 className="text-[13px] font-bold leading-tight text-ink">{p.nama}</h4>
              <p className="mt-0.5 text-[12px] leading-snug text-charcoal">{p.fungsi}</p>
            </div>
          </div>
        </div>
      ))}
    </div>

    {/* Tombol peta resmi */}
    {lokasi.petaResmiUrl && (
      <a
        href={lokasi.petaResmiUrl}
        target="_blank"
        rel="noopener noreferrer"
        className="mt-3 flex items-center justify-center gap-2 rounded-xl border border-primary/30 bg-primary/5 px-4 py-3 text-[13px] font-semibold text-primary transition-colors hover:bg-primary/10"
      >
        <IconPeta className="h-4 w-4" />
        Lihat Denah Resmi (alharamain.gov.sa)
      </a>
    )}

    {/* Disclaimer sumber */}
    <p className="mt-2 text-center font-mono text-[9px] uppercase tracking-wider text-mute">
      Sumber: General Authority for the Care of the Two Holy Mosques
    </p>
  </section>
)}
```

Pastikan `IconPeta` sudah di-import di LokasiDetail.tsx.
