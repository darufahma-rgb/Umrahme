## LANGKAH 2 — Tambah `className` ke elemen JSX

Tambahkan className berikut ke elemen yang tepat. **Jangan ubah style yang sudah ada**, hanya tambah `className`.

### A. Hero CTA buttons wrapper
Cari `motion.div` yang punya:
```
style={{ display: 'flex', gap: 12, justifyContent: 'center', flexWrap: 'wrap', marginBottom: 26 }}
```
Tambah: `className="lp-hero-btns"`

### B. Hero social proof wrapper
Cari `motion.div` yang punya:
```
style={{ display: 'flex', alignItems: 'center', gap: 10, justifyContent: 'center', color: '#fff', fontSize: 14, fontWeight: 600 }}
```
Tambah: `className="lp-hero-social"`

### C. Floating card ke-3 (Panduan Ibadah) — sembunyikan di mobile
Card ke-3 adalah `motion.div` yang di dalamnya ada `{i === 2 && (`. Tambahkan `className={i === 2 ? 'lp-float-card-mid' : ''}` ke `motion.div` tersebut.

### D. Partners marquee wrapper
Cari `div` yang punya:
```
style={{ padding: '90px 0 50px', borderBottom: ...
```
Tambah: `className="lp-marquee-wrap lp-section-sm"`

### E. Bento right column wrapper
Cari `div` yang punya:
```
style={{ display: 'flex', flexDirection: 'column', gap: 18 }}
```
*(yang ada di dalam section bento, bukan di tempat lain)*
Tambah: `className="lp-bento-right"`

### F. White-label phones wrapper
Cari `div` yang punya:
```
style={{ display: 'flex', gap: 18, justifyContent: 'center', flexWrap: 'wrap', overflow: 'hidden', padding: '10px 10px 20px' }}
```
Tambah: `className="lp-wl-phones"`

### G. Pricing card gelap
Cari `div` yang punya:
```
style={{ background: C.ink, color: '#fff', borderRadius: 28, padding: 40, position: 'relative', overflow: 'hidden', height: '100%', display: 'flex', flexDirection: 'column' }}
```
Tambah: `className="lp-price-card"`

### H. Angka "10" di pricing
Cari:
```jsx
<span style={{ fontSize: 64, fontWeight: 800, lineHeight: 1, color: C.accent }}>10</span>
```
Tambah: `className="lp-price-num"`

### I. Kalkulator card (sembunyikan di mobile)
Cari `<Reveal delay={0.08}>` yang di dalamnya ada teks "Kalkulator Biaya".
Tambah `className="lp-calc-hide"` ke `Reveal` tersebut:
```jsx
<Reveal delay={0.08} className="lp-calc-hide">
```

### J. CTA box gradient
Cari `div` yang punya:
```
style={{ background: `linear-gradient(165deg, ${C.primary}, ${C.primaryDark})`, color: '#fff', borderRadius: 32, padding: '64px 40px'...
```
Tambah: `className="lp-cta-box"`

### K. CTA buttons wrapper di dalam CTA box
Cari `div` yang punya:
```
style={{ display: 'flex', gap: 14, justifyContent: 'center', flexWrap: 'wrap', position: 'relative' }}
```
Tambah: `className="lp-cta-btns"`

### L. Semua section utama
Tambah `className="lp-section"` ke semua `<section>` dan wrapper `<div>` yang punya `padding: '90px 0'` atau `padding: '90px 0'` sebagai value style-nya. Ini mencakup:
- Section ABOUT/BENTO
- Section PAIN POINTS (`style={{ background: C.soft, padding: '90px 0' }}`)
- Section FEATURES (`id="fitur"`)
- Section HOW IT WORKS (`id="cara"`)
- Section WHITE-LABEL
- Section PRICING (`id="harga"`)
- Section TESTIMONIALS
- Section FAQ
- Section FINAL CTA (`id="kontak"` — ini punya `padding: '90px 24px'`)

### M. Semua inner container (max-width: 1180)
Cari semua `div` yang punya:
```
style={{ maxWidth: 1180, margin: '0 auto', padding: '0 24px' }}
```
Tambah: `className="lp-inner"` ke semua yang ditemukan.

### N. Footer links wrapper
Di dalam footer, wrap ketiga div kolom (Produk, Perusahaan, Legal) dengan satu `<div className="lp-foot-links">` baru.
