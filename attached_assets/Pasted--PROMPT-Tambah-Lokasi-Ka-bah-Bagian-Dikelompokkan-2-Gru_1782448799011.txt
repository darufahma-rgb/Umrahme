# PROMPT: Tambah Lokasi Ka'bah + Bagian Dikelompokkan 2 Grup (Rapi)

**File terkait:**
- `artifacts/umrahme/src/types/index.ts` (tambah tipe `bagianGrup`)
- `artifacts/umrahme/src/data/lokasi.ts` (tambah objek Ka'bah)
- `artifacts/umrahme/src/pages/LokasiDetail.tsx` (render 2 grup)

Tambahkan lokasi Ka'bah. Bagian-bagiannya JANGAN ditampilkan terpisah jadi banyak kartu — kelompokkan jadi **2 grup** saja: "4 Sudut (Rukun) Ka'bah" dan "Bagian Penting Lainnya". Tiap grup = 1 kartu berisi daftar ringkas.

**PENTING soal karakter:** tulis karakter Arab (ﷺ, الكعبة, dll), apostrof (’), dan em-dash (—) LANGSUNG sebagai karakter asli. JANGAN pakai escape `\uXXXX` apalagi double-backslash `\\uXXXX` — itu menyebabkan teks tampil mentah/ngawur. File `lokasi-kabah-grup.ts` sudah memakai karakter langsung yang benar; salin apa adanya.

## LANGKAH 1 — Tambah tipe di `types/index.ts`

```ts
export interface BagianKabahItem {
  nama: string;
  arab?: string;
  deskripsi: string;
}

export interface BagianKabahGrup {
  judul: string;              // mis. "4 Sudut (Rukun) Ka'bah"
  items: BagianKabahItem[];
}

export interface Lokasi {
  // ... field yang sudah ada ...
  diagramBagian?: string;           // path 1 diagram visual semua bagian Ka'bah
  bagianGrup?: BagianKabahGrup[];   // untuk Ka'bah: bagian dikelompokkan
}
```

## LANGKAH 2 — Tambah objek Ka'bah di `data/lokasi.ts`
Salin objek lengkap dari file `lokasi-kabah-grup.ts` ke dalam array `daftarLokasi` (letakkan di kelompok lokasi Makkah). Field `bagianGrup` berisi 2 grup:
1. **4 Sudut (Rukun) Ka'bah:** Rukun Hajar Aswad, Rukun Yamani, Rukun Iraqi, Rukun Syami.
2. **Bagian Penting Lainnya:** Multazam, Maqam Ibrahim, Hijir Ismail, Pintu Ka'bah, Mizab Ar-Rahman, Kiswah, Syadzarwan.

## LANGKAH 3 — Render 2 grup di `LokasiDetail.tsx`

Tampilkan hanya jika `lokasi.bagianGrup` ada. Mulai dengan SATU diagram visual (jika `diagramBagian` ada), lalu 2 kartu grup berisi daftar teks ringkas (TANPA foto per item). Letakkan setelah section Sejarah.

```tsx
{lokasi.bagianGrup && lokasi.bagianGrup.length > 0 && (
  <section className="mt-6">
    <div className="mb-3">
      <p className="font-mono text-[8.5px] uppercase tracking-[0.18em] text-mute">Rincian</p>
      <h3 className="mt-0.5 text-[15px] font-bold text-ink">Bagian-Bagian Ka’bah</h3>
    </div>

    {/* Satu diagram visual semua bagian */}
    {lokasi.diagramBagian && (
      <div className="mb-3 overflow-hidden rounded-2xl border border-hairline bg-white">
        <img src={lokasi.diagramBagian} alt="Diagram bagian-bagian Ka’bah" className="w-full" loading="lazy" />
      </div>
    )}

    <div className="space-y-3">
      {lokasi.bagianGrup.map((grup, gi) => (
        <div key={gi} className="rounded-2xl border border-hairline bg-surface-card p-4 shadow-drop-soft">
          {/* Judul grup */}
          <div className="mb-2 flex items-center gap-2">
            <span className="h-4 w-1 rounded-full bg-gold" aria-hidden />
            <h4 className="text-[13.5px] font-bold text-ink">{grup.judul}</h4>
          </div>

          {/* Daftar item: baris ringkas teks, dipisah garis halus (TANPA foto) */}
          <div className="divide-y divide-hairline">
            {grup.items.map((it, ii) => (
              <div key={ii} className="py-2.5 first:pt-0 last:pb-0">
                <div className="flex items-baseline justify-between gap-3">
                  <h5 className="text-[13px] font-semibold text-ink">{it.nama}</h5>
                  {it.arab && <span className="flex-none font-arab text-[14px] text-gold" dir="rtl">{it.arab}</span>}
                </div>
                <p className="mt-0.5 text-[12px] leading-snug text-charcoal">{it.deskripsi}</p>
              </div>
            ))}
          </div>
        </div>
      ))}
    </div>
  </section>
)}
```

Hasil: satu diagram visual di atas yang menunjukkan SEMUA bagian sekaligus, lalu 2 kartu (Sudut & Bagian Lain) berisi daftar teks ringkas. Tidak ada foto kecil per item.
