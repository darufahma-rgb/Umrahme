Kamu adalah senior product engineer dan UI/UX designer. Tolong update project Umrahme ini supaya dashboard jamaah terasa seperti aplikasi resmi dari travel yang benar-benar menemani perjalanan umrah jamaah, bukan sekadar aplikasi doa atau menu fitur.

Konteks project:

* App utama ada di `artifacts/umrahme`
* Dashboard utama jamaah ada di `src/pages/Beranda.tsx`
* Route utama ada di `src/App.tsx`
* Auth jamaah ada di `src/context/AuthContext.tsx`
* Tipe data ada di `src/types/index.ts`
* Data jamaah/demo login ada di `src/data/jamaah.ts`
* Agenda lengkap sudah ada di `src/pages/AgendaLengkap.tsx`
* Dashboard sudah mulai memakai data agenda dari Supabase table `agenda_items`
* Jangan hapus fitur lama.
* Jangan rombak total visual lama.
* Fokus utama: ubah flow dashboard agar terasa seperti travel companion.

Tujuan utama:
Dashboard harus menjawab 5 pertanyaan jamaah:

1. Saya ikut travel apa?
2. Saya sekarang di fase perjalanan apa?
3. Hari ini saya harus ngapain?
4. Ada pengumuman penting apa dari travel?
5. Kalau bingung atau tersesat, saya harus hubungi siapa?

Formula dashboard baru:

IDENTITAS → ARAHAN HARI INI → INFO PENTING → BANTUAN → IBADAH

Bukan:

MENU → MENU → MENU → MENU

FILE DATA DUMMY
===============

Buat file:

`src/data/travelCompanion.ts`

Isi data dan helper untuk:

* operational info
* announcement
* focus berdasarkan fase
* whatsapp link

Data minimal:

```ts
import type { Fase } from '../types';

export type TravelOperationalInfo = {
  groupCode: string;
  busNumber: string;
  roomNumber: string;
  hotelMakkah: string;
  hotelMadinah: string;
  meetingPoint: string;
  guideName: string;
  guideRole: string;
  guideWhatsapp: string;
  travelWhatsapp: string;
  emergencyNote: string;
};

export type TravelAnnouncement = {
  id: string;
  label: string;
  title: string;
  content: string;
  important?: boolean;
  publishedAt: string;
};

const DEFAULT_OPERATIONAL_INFO: TravelOperationalInfo = {
  groupCode: 'Rombongan A',
  busNumber: 'Bus 3',
  roomNumber: '804',
  hotelMakkah: 'Hotel Makkah — diisi travel',
  hotelMadinah: 'Hotel Madinah — diisi travel',
  meetingPoint: 'Lobby hotel / titik kumpul dari pembimbing',
  guideName: 'Ust. Pembimbing',
  guideRole: 'Pembimbing Rombongan',
  guideWhatsapp: '6281234567890',
  travelWhatsapp: '6281234567890',
  emergencyNote: 'Jika tersesat, tetap tenang. Hubungi pembimbing dan tunjukkan Kartu Jamaah Digital.',
};

const DEFAULT_ANNOUNCEMENTS: TravelAnnouncement[] = [
  {
    id: 'announcement-1',
    label: 'Penting',
    title: 'Perhatikan titik kumpul hari ini',
    content:
      'Jamaah dimohon berkumpul sesuai arahan pembimbing. Bawa sandal, tas kecil, kartu jamaah, dan tetap bersama rombongan.',
    important: true,
    publishedAt: new Date().toISOString(),
  },
];

export function getOperationalInfo(_tenantId?: string | null): TravelOperationalInfo {
  return DEFAULT_OPERATIONAL_INFO;
}

export function getLatestAnnouncement(_tenantId?: string | null): TravelAnnouncement | null {
  return DEFAULT_ANNOUNCEMENTS[0] ?? null;
}

export function getFocusByFase(fase: Fase) {
  if (fase === 'tanah-suci') {
    return {
      title: 'Fokus ibadah hari ini',
      description:
        'Ikuti arahan pembimbing, buka panduan ibadah, dan gunakan counter saat tawaf atau sa’i.',
      ctaLabel: 'Mulai Ibadah',
      ctaTo: '/ibadah/navigator',
    };
  }

  if (fase === 'selesai') {
    return {
      title: 'Jaga kenangan perjalanan',
      description:
        'Lengkapi jurnal, simpan sertifikat, dan terus jaga amal setelah pulang.',
      ctaLabel: 'Buka Jurnal',
      ctaTo: '/profil/jurnal',
    };
  }

  return {
    title: 'Persiapkan keberangkatan',
    description:
      'Lengkapi checklist, dokumen, perlengkapan, dan pahami tata cara umrah sebelum berangkat.',
    ctaLabel: 'Lengkapi Persiapan',
    ctaTo: '/profil/persiapan',
  };
}

export function whatsappLink(
  phone: string,
  message = 'Assalamu’alaikum, saya butuh bantuan selama perjalanan umrah.'
): string {
  const clean = phone.replace(/\D/g, '');
  const normalized = clean.startsWith('0') ? `62${clean.slice(1)}` : clean;
  return `https://wa.me/${normalized}?text=${encodeURIComponent(message)}`;
}
```

==================================================
ACCEPTANCE CRITERIA
===================

Perubahan dianggap berhasil jika:

1. Dashboard langsung terasa sebagai aplikasi travel.
2. Jamaah melihat nama travel dan identitas perjalanan.
3. Jamaah melihat rombongan, bus, kamar, hotel.
4. Jamaah melihat fokus perjalanan sesuai fase.
5. Jamaah melihat pengumuman penting dari travel.
6. Jamaah bisa langsung WhatsApp/telepon pembimbing.
7. Jamaah bisa membuka kartu jamaah digital.
8. Fitur lama tetap ada dan tidak rusak.
9. Route `/profil/kartu` bisa dibuka.
10. App tetap bisa build tanpa TypeScript error.
11. Jalankan `npm run build` dan perbaiki error sampai berhasil.

==================================================
CATATAN PENTING
===============

Jangan mulai dari database/admin dulu. Untuk tahap ini buat flow dashboard hidup dengan data lokal/fallback dulu. Setelah UI dan flow terasa benar, nanti baru disambungkan ke Supabase dan admin travel.

Prioritas pengalaman:
Jamaah buka app → langsung tahu ini aplikasi dari travelnya → tahu agenda/fokus hari ini → tahu pengumuman penting → tahu siapa yang harus dihubungi → bisa buka panduan ibadah.

Jangan jadikan dashboard hanya kumpulan menu.
Jadikan dashboard sebagai pusat komando perjalanan jamaah.
