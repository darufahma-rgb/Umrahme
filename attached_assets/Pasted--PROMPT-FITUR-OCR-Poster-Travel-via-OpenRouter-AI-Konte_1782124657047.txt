# PROMPT FITUR — OCR Poster Travel via OpenRouter AI

> Konteks: admin panel di `/admin` sudah bisa kelola tenant. Fitur baru: admin upload foto/gambar poster keberangkatan dari travel agency, AI (via OpenRouter) membaca dan mengekstrak data dari poster tersebut, lalu hasil ekstraksi otomatis mengisi form tenant (nama travel, tanggal keberangkatan, hotel, kontak pembimbing, dll) tanpa harus ketik manual.
>
> Arsitektur: karena `OPENROUTER_API_KEY` tidak boleh di-expose ke browser, buat Vercel API Route sebagai proxy. Frontend upload gambar → Vercel API Route → OpenRouter Vision API → hasil JSON kembali ke frontend → auto-fill form.

## 1. Buat Vercel API Route `api/ai-ocr.ts`

Buat file `api/ai-ocr.ts` di ROOT project (bukan di `artifacts/umrahme/`), ini akan jadi serverless function Vercel:

```ts
import type { VercelRequest, VercelResponse } from '@vercel/node';

export default async function handler(req: VercelRequest, res: VercelResponse) {
  if (req.method !== 'POST') {
    return res.status(405).json({ error: 'Method not allowed' });
  }

  const { imageBase64, mimeType } = req.body as {
    imageBase64: string;
    mimeType: string;
  };

  if (!imageBase64 || !mimeType) {
    return res.status(400).json({ error: 'imageBase64 dan mimeType wajib diisi.' });
  }

  const apiKey = process.env.OPENROUTER_API_KEY;
  if (!apiKey) {
    return res.status(500).json({ error: 'OPENROUTER_API_KEY belum dikonfigurasi.' });
  }

  const prompt = `Kamu adalah sistem ekstraksi data dari poster/brosur perjalanan umrah.

Analisis gambar ini dan ekstrak informasi berikut dalam format JSON.
Jika data tidak ditemukan, isi dengan null.
Jangan tambahkan penjelasan apapun, hanya kembalikan JSON murni.

Format JSON yang diharapkan:
{
  "nama_travel": string | null,
  "tanggal_keberangkatan": string | null,  // format: YYYY-MM-DD
  "tanggal_kepulangan": string | null,      // format: YYYY-MM-DD
  "hotel_makkah": string | null,
  "hotel_madinah": string | null,
  "guide_name": string | null,
  "guide_whatsapp": string | null,
  "tour_leader_name": string | null,
  "tour_leader_whatsapp": string | null,
  "meeting_point": string | null,
  "catatan": string | null
}`;

  try {
    const response = await fetch('https://openrouter.ai/api/v1/chat/completions', {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${apiKey}`,
        'Content-Type': 'application/json',
        'HTTP-Referer': process.env.APP_URL ?? 'https://umrahme.app',
        'X-Title': 'Umrahme Admin',
      },
      body: JSON.stringify({
        model: 'google/gemini-flash-1.5',
        messages: [
          {
            role: 'user',
            content: [
              { type: 'text', text: prompt },
              {
                type: 'image_url',
                image_url: {
                  url: `data:${mimeType};base64,${imageBase64}`,
                },
              },
            ],
          },
        ],
        max_tokens: 1000,
        temperature: 0.1,
      }),
    });

    if (!response.ok) {
      const err = await response.text();
      return res.status(502).json({ error: `OpenRouter error: ${err}` });
    }

    const data = await response.json();
    const content = data.choices?.[0]?.message?.content ?? '';

    // Parse JSON dari response — bersihkan markdown code block jika ada
    const cleaned = content.replace(/```json|```/g, '').trim();
    const extracted = JSON.parse(cleaned);

    return res.status(200).json({ ok: true, data: extracted });
  } catch (err) {
    return res.status(500).json({ error: String(err) });
  }
}
```

## 4. Tambah fitur OCR di `AdminTenantForm.tsx`

Tambahkan section "Import dari Poster" di PALING ATAS form (sebelum field Kode Aktivasi), hanya muncul saat `isNew` ATAU saat form sedang kosong — ini tool bantu pengisian awal, bukan wajib.

### State baru
```ts
const [posterFile, setPosterFile] = useState<File | null>(null);
const [ocrLoading, setOcrLoading] = useState(false);
const [ocrError, setOcrError] = useState('');
const [ocrResult, setOcrResult] = useState<Record<string, string | null> | null>(null);
```

### Fungsi handler OCR
```ts
async function handleOcrPoster(file: File) {
  setOcrLoading(true);
  setOcrError('');

  try {
    // Convert file ke base64
    const base64 = await new Promise<string>((resolve, reject) => {
      const reader = new FileReader();
      reader.onload = () => resolve((reader.result as string).split(',')[1]);
      reader.onerror = reject;
      reader.readAsDataURL(file);
    });

    const response = await fetch('/api/ai-ocr', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ imageBase64: base64, mimeType: file.type }),
    });

    const result = await response.json();
    if (!result.ok) throw new Error(result.error);

    const d = result.data;
    setOcrResult(d);

    // Auto-fill form fields jika nilainya ada
    if (d.nama_travel) setNamaTravel(d.nama_travel);
    if (d.tanggal_keberangkatan) setTanggalKeberangkatan(d.tanggal_keberangkatan);
    if (d.tanggal_kepulangan) setTanggalKepulangan(d.tanggal_kepulangan);
    if (d.hotel_makkah) setHotelMakkah(d.hotel_makkah);
    if (d.hotel_madinah) setHotelMadinah(d.hotel_madinah);
    if (d.guide_name) setGuideName(d.guide_name);
    if (d.guide_whatsapp) setGuideWhatsapp(d.guide_whatsapp);
    if (d.tour_leader_name) setTourLeaderName(d.tour_leader_name);
    if (d.tour_leader_whatsapp) setTourLeaderWhatsapp(d.tour_leader_whatsapp);
    if (d.meeting_point) setMeetingPoint(d.meeting_point);

  } catch (err) {
    setOcrError(err instanceof Error ? err.message : 'Gagal membaca poster.');
  } finally {
    setOcrLoading(false);
  }
}
```

### UI section OCR (letakkan sebelum form fields)
```tsx
{/* ── Import dari Poster ── */}
<div className="rounded-2xl border-2 border-dashed border-primary/30 bg-primary/3 p-5">
  <p className="font-mono text-[10px] uppercase tracking-widest text-primary mb-1">
    ✨ Import dari Poster
  </p>
  <p className="text-[12px] text-charcoal mb-3">
    Upload foto poster/brosur keberangkatan — AI akan membaca dan mengisi form secara otomatis.
  </p>

  <input
    type="file"
    accept="image/png,image/jpeg,image/webp"
    onChange={e => {
      const file = e.target.files?.[0];
      if (file) { setPosterFile(file); handleOcrPoster(file); }
    }}
    className="hidden"
    id="poster-upload"
  />

  <label htmlFor="poster-upload" className="cursor-pointer inline-flex items-center gap-2 px-4 py-2 rounded-xl bg-primary text-white text-[12px] font-semibold transition-all hover:bg-primary-deep">
    {ocrLoading ? 'Membaca poster...' : '📎 Pilih Foto Poster'}
  </label>

  {ocrLoading && (
    <p className="mt-2 text-[11px] text-primary animate-pulse">
      AI sedang membaca poster, mohon tunggu...
    </p>
  )}

  {ocrError && (
    <p className="mt-2 text-[11px] text-red-500">{ocrError}</p>
  )}

  {ocrResult && !ocrLoading && (
    <p className="mt-2 text-[11px] text-green-600">
      ✓ Data berhasil diekstrak dan form telah diisi otomatis. Periksa dan sesuaikan jika perlu.
    </p>
  )}
</div>
```
