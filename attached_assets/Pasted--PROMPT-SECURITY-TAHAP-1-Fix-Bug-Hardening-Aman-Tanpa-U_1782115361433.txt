## 2. Fix upload logo — tolak SVG, validasi MIME type

File: `src/pages/admin/AdminTenantForm.tsx`

### 2a. Ubah `accept` di input file (baris 489):
```tsx
// SEBELUM:
accept="image/png,image/svg+xml,image/jpeg,image/webp"

// SESUDAH:
accept="image/png,image/jpeg,image/webp"
```

### 2b. Tambahkan validasi MIME type di `handleLogoChange` (setelah cek `file.size`):
```ts
const allowedTypes = ['image/png', 'image/jpeg', 'image/webp'];
if (!allowedTypes.includes(file.type)) {
  setError('Logo harus berupa PNG, JPG, atau WEBP. SVG tidak diizinkan.');
  return;
}
```

const ext = file.type === 'image/png' ? 'png' : file.type === 'image/webp' ? 'webp' : 'jpg';

## 3. Validasi input di form admin

File: `src/pages/admin/AdminTenantForm.tsx`

### 3a. Validasi warna hex sebelum submit:
Tambahkan helper di atas komponen:
```ts
function isValidHex(val: string) {
  return /^#[0-9A-Fa-f]{6}$/.test(val);
}
```
Di `handleSubmit`, sebelum submit payload, cek:
```ts
if (!isValidHex(primaryColor)) {
  setError('Warna primary harus berupa hex valid, contoh: #0ea5e9');
  return;
}
```

### 3b. Normalisasi nomor WhatsApp:
Tambahkan helper:
```ts
function normalizePhone(val: string) {
  return val.replace(/[^\d+]/g, '');
}
```
Terapkan saat mengisi state `guideWhatsapp` dan `tourLeaderWhatsapp`:
```tsx
onChange={(e) => setGuideWhatsapp(normalizePhone(e.target.value))}
```

### 3c. Batasi panjang field (terapkan via `maxLength` di input masing-masing):
- Nama travel: `maxLength={80}`
- Judul pengumuman: `maxLength={120}`
- Isi pengumuman: `maxLength={1000}` (textarea)
- Hotel Makkah/Madinah: `maxLength={120}`
- Meeting point: `maxLength={160}`
- Nama pembimbing/tour leader: `maxLength={80}`
