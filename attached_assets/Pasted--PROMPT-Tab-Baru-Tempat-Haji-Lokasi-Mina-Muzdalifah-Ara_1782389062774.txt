# PROMPT — Tab Baru "Tempat Haji" + Lokasi Mina, Muzdalifah, Arafah dll

## Konteks
File data: `artifacts/umrahme/src/data/lokasi.ts`
File tampilan: `artifacts/umrahme/src/pages/Peta.tsx`

Tambahkan kategori/tab baru **"Tempat Haji"** untuk lokasi inti pelaksanaan haji
(Mina, Muzdalifah, Arafah) beserta masjid pentingnya (Namira, Khayf, Jamarat).
Jabal Rahmah yang sudah ada juga dipindah ke kategori ini.

Koordinat sudah diverifikasi via riset (Wikipedia, Grokipedia, Madain Project, makkahlive.net).

---

## LANGKAH 1 — Tambah Tab & Kategori "Tempat Haji"

### 1a. Update type `KotaFilter` & fungsi `lokasiByKota`

File: `artifacts/umrahme/src/data/lokasi.ts`

Cari (kondisi saat ini, sudah termasuk 'Lainnya' dari prompt sebelumnya):
```ts
export type KotaFilter = 'Makkah' | 'Madinah' | 'Lainnya';

export function lokasiByKota(kota: KotaFilter): Lokasi[] {
  return daftarLokasi.filter((l) => {
    if (kota === 'Makkah') return l.kota === 'Makkah' || l.kota === 'Arafah';
    if (kota === 'Madinah') return l.kota === 'Madinah';
    // 'Lainnya' = semua yang BUKAN Makkah/Arafah/Madinah
    return l.kota !== 'Makkah' && l.kota !== 'Arafah' && l.kota !== 'Madinah';
  });
}
```

Ganti jadi:
```ts
export type KotaFilter = 'Makkah' | 'Madinah' | 'Haji' | 'Lainnya';

// Kota yang termasuk kategori "Tempat Haji"
const KOTA_HAJI = ['Mina', 'Muzdalifah', 'Arafah'];

export function lokasiByKota(kota: KotaFilter): Lokasi[] {
  return daftarLokasi.filter((l) => {
    if (kota === 'Makkah') return l.kota === 'Makkah';
    if (kota === 'Madinah') return l.kota === 'Madinah';
    if (kota === 'Haji') return KOTA_HAJI.includes(l.kota);
    // 'Lainnya' = bukan Makkah/Madinah/Haji
    return l.kota !== 'Makkah' && l.kota !== 'Madinah' && !KOTA_HAJI.includes(l.kota);
  });
}
```

> Perhatikan: 'Arafah' DIPINDAH dari grup Makkah ke grup Haji. Jabal Rahmah (kota='Arafah')
> otomatis ikut pindah ke tab "Tempat Haji".

### 1b. Tambah tab di Peta.tsx

File: `artifacts/umrahme/src/pages/Peta.tsx`

Cari:
```ts
const tabs: { id: KotaFilter; label: string }[] = [
  { id: 'Makkah', label: 'Makkah' },
  { id: 'Madinah', label: 'Madinah' },
  { id: 'Lainnya', label: 'Lainnya' },
];
```

Ganti jadi:
```ts
const tabs: { id: KotaFilter; label: string }[] = [
  { id: 'Makkah', label: 'Makkah' },
  { id: 'Madinah', label: 'Madinah' },
  { id: 'Haji', label: 'Tempat Haji' },
  { id: 'Lainnya', label: 'Lainnya' },
];
```

> Dengan 4 tab, label bisa agak sempit di mobile. Kalau perlu, kecilkan sedikit font tab
> dari `text-[13px]` ke `text-[12px]` agar "Tempat Haji" tidak terpotong. Cek visual setelahnya.

## Catatan Penting
- Koordinat terverifikasi:
  - Padang Arafah: 21.3550, 39.9839
  - Mina: 21.4133, 39.8933
  - Muzdalifah: 21.3925, 39.9378 (Wikipedia DMS 21°23'33"N 39°56'16"E)
  - Masjid Namira: 21.3547, 39.9728 (~1.5km dari Jabal Rahmah)
  - Masjid Khayf: 21.4108, 39.8867
  - Jamarat: 21.4222, 39.8728
- Jabal Rahmah (kota='Arafah') OTOMATIS pindah ke tab "Tempat Haji" karena perubahan filter
- Field `gambar: ''` dikosongkan — bisa diisi nanti kalau ada foto
- Field `kota` bertipe `string` — "Mina", "Muzdalifah", "Arafah" otomatis valid
- Pastikan karakter Arab & emoji ﷺ/عليه السلام ter-encode UTF-8 benar
- Jangan ubah lokasi lain yang sudah ada (selain perubahan filter yang memindah Arafah)
