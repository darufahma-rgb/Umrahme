## BAGIAN 3 — Perbaiki Format URL Google Maps

File: `artifacts/umrahme/src/pages/LokasiDetail.tsx`

Format URL saat ini memakai sintaks `query=lat,lng(nama)` yang kadang bikin pin tidak tepat.
Ganti jadi format resmi Google Maps yang lebih reliable.

Cari:
```ts
function gmapsUrl(lat: number, lng: number, nama: string): string {
  return `https://www.google.com/maps/search/?api=1&query=${lat},${lng}(${encodeURIComponent(nama)})`;
}
```

Ganti jadi:
```ts
function gmapsUrl(lat: number, lng: number, _nama: string): string {
  // Format resmi Google Maps Search API — pin tepat di koordinat
  return `https://www.google.com/maps/search/?api=1&query=${lat}%2C${lng}`;
}
```

> Catatan: parameter `nama` tetap ada di signature (jadi `_nama`) supaya pemanggilan
> di baris `gmapsUrl(lokasi.koordinat.lat, lokasi.koordinat.lng, lokasi.nama)` tidak error.
> Kalau mau tetap kirim nama sebagai konteks, alternatifnya:
> `query=${encodeURIComponent(nama)}&center=${lat},${lng}` — TAPI pakai koordinat murni
> lebih aman supaya tidak salah lokasi karena nama ambigu.

## Catatan
- Tipe `koordinat` adalah `{ lat: number; lng: number }` — pastikan tetap number, bukan string
- Jangan ubah `jarakKm`, `sejarah`, atau field lain
- Hanya 3 koordinat yang berubah (Jabal Tsur, Qiblatain, Quba) + 1 perbaikan fungsi URL
